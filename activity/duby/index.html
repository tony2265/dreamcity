我們是誰？

夢想之都工作室（DreamCity Studio）

成立於2015年，團隊成員多以學生為主，來自臺灣、香港與澳門等地，為跨足兩岸三地的線上學生社群組織，一直以來，我們深耕於Minecraft地圖製作，致力於創作並推廣Minecraft；在從事創作的過程，也藉此磨練專長、技術，與團員共同成長。 夢想之都擁有豐富的Minecraft地圖創作經驗，希望帶著我們的經驗與能力，踏入Vtuber的大家庭，拓展團隊新視野。

我們在找誰？

能夠一同創作成長的夥伴

我們所希望邀請的不僅僅只是Vtuber的幕後中之人，更是能夠與夢想之都工作室一同創作與成長的夥伴，您所擔任的角色基礎是由團隊規劃而誕生，而我們現在正在尋找這個團隊中最核心的一員，一同完善這個角色。

我們需要請您具備以下幾項要點：

  1. 對此領域有強烈企圖心，能夠與我們一同展開熱烈討論，未來這是屬於你的舞台，需要你具備有熱忱與創意
  2. 擁有充足的時間能夠進行發想、創作、直播與討論
  3. 擁有英文或日語溝通及閱讀能力，使非華語地區的觀眾也能一同參與
  4. 具有與團隊妥善溝通的能力，溝通為的是幫助我們更能相互了解

相關規範

你得具備什麼？

  • 您必須在兩年內都能夠持續地製作影片和直播
  • 您必須能夠每週製作影片和直播至少 3 次 (正常情況下)
  • 您必須能夠每週與團隊至少開1次會議
  • 您必須年滿 18 歲
  • 您必須擁有能不被打擾的直播環境與可流暢進行遊戲與直播的設備

我們提供什麼？？

  • 美術資源全數由夢都包辦（Live2D、背景圖、Logo字、待機動畫...）
  • 全新Minecraft帳號
  • 相關設備（包含捕捉設備、基本錄音設備等）
  • 精華影片剪輯
  • 需於出道前與我們一同針對角色進行討論和規劃（夢都與中之的關係傾向是夥伴）
  • 可討論的收益分成
  • 由專人協助各項企劃進行
    • YouTube頻道企劃
    • 社群經營規劃
    • 宣傳曝光規劃

申請者要點

務必達成條件
  • 對此領域有熱忱和想法，且能與團隊、與觀眾良好溝通
  • 願意接觸和學習全新事物
  • 愛聊天，平時有發布或經營社群媒體習慣
  • 能夠帶動觀眾情緒和直播氣氛
  • 良好團隊合作能力，且能清楚有條理的表達想法和情緒
  • 能夠投入演繹一個角色及遵守基礎設定
  • 能夠配合每週進行直播活動，且願意與團隊有長時間合作
  • 擁有遊玩Minecraft之遊戲經驗
  • 擁有遊玩TRPG之遊戲經驗
加分條件
  1. 歡迎提供可檢視的作品集（包括但不限於：影片、音樂、繪圖等作品）
  2. 擁有正在經營中的社群（包括但不限於：YouTube、Twitch、Twitter、Facebook、Plurk、Discord等）
  3. 擁有不同面向之特色能力（包括但不限於：唱歌、跳舞、樂器、插畫、設計、剪輯、動畫製作、程式開發等）
  4. 能用英日文流暢表達
  5. 具備演藝活動之相關經驗
  6. 其他你認為特殊的才能皆可提出加分
